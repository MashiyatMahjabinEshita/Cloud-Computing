# Cloud-Computing

In this repository, I have added the contents of my cloud computing course. It contains,
- Intro to KVM.
- Intro to Docker.
- Intro to RabbitMQ.
- Intro to OpenStackSwift and Its Installation and Basics.


Here are two reference books you might read to expand your knowledge, suggested by my course instructor:
- Cloud Computing Theory And Practice by Dan C.. Marinescu.
- Cloud Computing: from beginning to end by Ray J. Rafals.
